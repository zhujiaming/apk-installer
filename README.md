### Apk Installer

  一款pc端使用的apk安装器，可以通过双击Apk直接将Apk安装到当前已连接Window/MacOs的Andorid手机设备上。

### for

  1. 同步支持windows、macOs，功能无差异；
  2. 拆包即用，无额外配置，简洁无额外功能；
  3. 自动关联为Apk文件默认打开方式，双击apk直接启动安装。；
  4. 支持多个设备同步一键安装；
  5. 安装状态日志提示；
